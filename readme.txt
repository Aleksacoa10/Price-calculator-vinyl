A simple price calculator for vinyl packages based on square meters, with an option to add 10% extra for cutting.

== Description ==

The Price Calculator plugin allows users to calculate the number of vinyl packages they need to purchase based on the square meters of their home or office. It also provides an option to add 10% extra to account for cutting.

**Features:**
* Calculate the number of vinyl packages based on square meters.
* Option to add 10% extra for cutting.
* Easy to use with a simple shortcode.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/price-calculator` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the shortcode `[price_calculator]` to display the calculator on any post or page.

== Frequently Asked Questions ==

= How do I use the shortcode? =

Simply place the `[price_calculator]` shortcode in any post or page where you want the calculator to appear.

= Can I customize the styles of the calculator? =

Yes, you can customize the styles by editing the CSS file located in `assets/css/style.css`.
